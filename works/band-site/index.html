バンドWebサイト

バンド活動用に制作したWebサイトです。HTML/CSSによるレイアウト設計、画像配置、レスポンシブ対応を行いました。

公開デモを見る 一覧へ戻る
制作形式
自主制作
実装体制
企画整理からHTML/CSS実装まで一人で担当
使用技術
HTML / CSS / Responsive

概要

ニュース、メンバー、リリース情報を掲載する静的サイトです。バンド活動の情報を1ページ内で見やすく確認できるように、見出し、画像、リンク導線を整理しました。

実装したこと

  • トップ、ニュース、メンバー、リリース情報のHTML構成
  • 画像を活かしたファーストビューと各セクションのレイアウト
  • スマホでも読みやすい余白とナビゲーションの調整
  • 公開用デモ向けの実在情報・外部リンクの整理

工夫した点

閲覧者がニュース、作品、問い合わせ導線を迷わず確認できるように、情報の順番と余白を意識しました。シンプルなHTML/CSS構成にして、後から告知やリリース情報を追加しやすい形にしています。
