在校届補助ツール

在校届作成を補助するために制作したWebアプリです。FlaskとSQLiteを用いて、入力内容の管理とPDF生成機能を実装しました。

使用技術
Python / Flask / SQLite / HTML / CSS / JavaScript

概要

学生の在校登録、教室変更、提出担当者設定、PDF出力を補助する小規模Webアプリです。Raspberry Pi上でのLAN内運用を想定し、日常的に使いやすい入力画面を目指して制作しました。

実装したこと

  • Flaskによるメイン画面と管理画面の実装
  • SQLiteを使った学生情報、教室情報、登録状態の管理
  • カード型UIによる登録・解除操作
  • 登録内容をもとにしたPDF生成処理
  • 公開用に実データを含まない静的匿名デモを作成

工夫した点

現場で素早く登録できるカード型UI、管理画面での名簿編集、PDF出力位置の設定分離など、運用後の修正しやすさを意識しました。公開デモでは、実データや内部情報を含めずに操作感だけを確認できるようにしています。
