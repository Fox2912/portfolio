SDGs診断Webコンテンツ

依頼を受けて制作した、JavaScriptを用いた診断型Webコンテンツです。回答内容に応じて結果を切り替えるロジックを実装しました。

公開デモを見る 一覧へ戻る
制作形式
依頼制作
実装体制
画面制作から診断ロジックまで一人で実装
使用技術
HTML / CSS / JavaScript / localStorage

概要

質問に回答するとタイプ別の結果画面へ遷移する診断コンテンツです。静的ファイルのみで動作し、GitHub Pagesなどでも公開しやすい構成です。

実装したこと

  • 質問画面、結果画面、キャラクター表示画面の作成
  • 回答内容に応じたスコア集計と条件分岐
  • 結果情報の保存と画面遷移の制御
  • ボタン操作を中心にしたインタラクティブなUI実装

工夫した点

回答の進み具合が分かるUIと、結果ごとの見せ方を分けることで、診断コンテンツらしい操作感を意識しました。依頼内容に合わせて、ユーザーが迷わず進められる導線にしています。
